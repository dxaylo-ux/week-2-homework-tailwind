Лоиҳаи Cootels як вебсайти муосири меҳмонхона мебошад, ки барои ҷалби сайёҳон
ва меҳмонон сохта шудааст. Ҳадафи асосии сайт пешниҳод намудани муҳити ором,
зебо ва наздик ба табиат барои истироҳати одамон мебошад.
Тарҳи сайт бо услуби минималистӣ ва рангорангии ором омода шуда,
ба корбар имкон медиҳад, ки маълумоти заруриро ба осонӣ пайдо намояд.
